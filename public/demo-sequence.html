Worked example — Sequence

This is the complete set of human inputs that plans a blank copy of the Sequence app — the ADHD-friendly household task sequencer — page by page and field by field, exactly as you'd type them into the wizard. It captures the product, the requirements behind every surface (editor, TV, login, sequencing, recurring, PWA, push), the locked stack decisions, milestones, risks, and the non-functional / scale needs.

Run it through the wizard and it generates the project's /docs structure plus a PLAN-INTAKE.json and an AI handoff prompt — the package a coding agent turns into the working app. (No household data is assumed; this plans a fresh, empty app.)
